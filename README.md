## O que é?

<p>Password Generator é um gerador de senhas</p>

## Como utilizar?

Para utilizar a ferramanta, basta seguir passa a passo a seguir:

- Copie o link abaixo e cole no seu navegador:
```
https://fabio-oliveira-martins.github.io/Password-Generator/
```
- Após carregar a página, aperte o botão preto escrito "Generate" para gerar uma senha aleatória.
- Por fim, ao lado do campo onde aparece a senha gerada, pressione o botão com o ícone de duas folhas de papel para poder copiar a senha.
